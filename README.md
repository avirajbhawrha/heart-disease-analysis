# ❤️ Heart Disease Analysis Project

 📌 Overview

This project focuses on analyzing heart disease data to understand patterns, relationships, and key factors that contribute to heart disease. The goal is to perform data analysis and build insights that can help in prediction and decision-making.



 📊 Dataset

The dataset contains medical attributes related to patients such as:

* Age
* Sex
* Chest Pain Type
* Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* Oldpeak
* ST Slope
* Heart Disease (Target)



🎯 Objectives

* Perform data cleaning and preprocessing
* Explore data using visualization
* Identify important features affecting heart disease
* Apply statistical and machine learning techniques
* Build predictive models


 🛠️ Technologies Used
